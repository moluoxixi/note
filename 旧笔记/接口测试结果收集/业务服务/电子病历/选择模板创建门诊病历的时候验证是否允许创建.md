## 请求参数:
``` json
{
  "hospCode": "NXRY",
  "patientId": "93587a7c8028404c85b09f8bc9803a7c",
  "orgCode": "",
  "stdRecordCode": "EMR020001",
  "regId": "1848171145803902977"
}
```
## 返回参数:
``` json
{
  "exception": null,
  "apiCode": null,
  "data": null,
  "Code": 1,
  "Message": "医院编码不能为空"
}
```
