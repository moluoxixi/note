# linux
-r 递归
-i 不区分大小写


cat 用于链接文件，基本功能是读取文件内容并将其输出到标准输出
grep 用于文本搜索，

| 命令                        | 说明                 |
| ------------------------- | ------------------ |
| `cat /etc/os-release`     | 查看系统版本信息，例如 Ubuntu |
| `cat 文件名`                 | 输出单个文件内容           |
| `grep 命令 "搜索内容" 文件名/文件夹名` | 在指定文件/文件夹中搜索指定内容   |
| `cat 文件名 \| grep "搜索内容"`  | 在指定文件中搜索内容         |
| `> 文件名`                   | 将内容输出到指定文件中        |
